##### iTV网关安装步聚 (非必须)

1. 下载秒开iTV网关操作系统ISO

2. 使用[U盘安装盘制作工具](https://rufus.ie/)，制作U盘启动盘

3. 使用U盘启动服务器，安装秒开iTV操作系统 （默认安装可以全部按回车[Enter]键)

4. 安装完成后，显示屏会显示iTV系统的管理口IP地址，把对应网口接入内网交换机

5. 在同一交换机，同网段，接入一台Windows电脑，打开浏览器，输入管理口IP地址

6. 进入管理配置WEB页，对iTV系统进行配置
